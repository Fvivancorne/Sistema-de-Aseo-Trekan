# Aseo Comunitario · Colegio Waldorf Trekan

App de gestión del calendario de aseo comunitario.

## Setup

### 1. Supabase — crear las tablas
- Ve a tu proyecto en supabase.com
- SQL Editor → New query
- Pega el contenido de `supabase_schema.sql` y ejecuta

### 2. Instalar dependencias y correr local
```bash
npm install
npm run dev
```

### 3. Deploy en Vercel
- Sube este repo a GitHub
- En vercel.com → New Project → importa el repo
- Vercel detecta Vite automáticamente
- Click Deploy — listo
